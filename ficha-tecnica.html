Especificaciones Técnicas

Información detallada para producción y montaje de presentaciones

Minimal Marimba combina la percusión clásica con la experimentación electrónica. Esta ficha técnica proporciona toda la información necesaria para productores, técnicos de sonido y organizadores de eventos que deseen programar nuestras presentaciones.

Formatos de Presentación

Marimba Acústica

  • Duración: 45-60 minutos
  • Espacio mínimo: 3m x 2m
  • Acústica: Preferentemente con buena reverberación natural

MENG-Âmok

  • Duración: 60-90 minutos
  • Tomas eléctricas: 2 independientes
  • Superficie: Mesa estable de 2m x 1m

Formato Mixto

  • Duración: 60-75 minutos
  • Espacio mínimo: 4m x 3m
  • Amplificación: 2 micrófonos para marimba

Cada formato puede adaptarse a las necesidades específicas del evento, con posibilidad de ajustar repertorio, duración y requerimientos técnicos según el espacio y contexto.

Requerimientos para Presentaciones

Condiciones Generales

Tiempo de montaje 45-60 minutos (dependiendo del formato)
Tiempo de desmontaje 50 minutos
Temperatura ambiente Control climático recomendado (18°-24°C)
Iluminación Posibilidad de ajuste según necesidades del programa

Condiciones Técnicas Específicas

  • Electricidad: Conexiones a tierra y estabilizadas para equipos electrónicos
  • Audio: Sistema de PA de calidad profesional con técnico de sonido
  • Personal de apoyo: Asistente para montaje/desmontaje de marimba (recomendado)
  • Personal de gira (opcional):
    • Artista principal
    • Técnico de sonido personal (según complejidad)
    • Asistente de instrumentos

Necesidades Técnicas para Grabación

Para Sesiones de Estudio

  • Microfonía especializada para marimba (par estéreo, micrófonos de condensador)
  • Espacio con acústica controlada
  • Posibilidad de grabación multicanal para posteriores mezclas
  • Monitoreo de referencia para sincronización con elementos pregrabados

Para proyectos de grabación específicos, se recomienda consultar previamente las necesidades particulares según el repertorio y formato a registrar. Disponemos de equipamiento propio pero es preferible conocer las características del estudio con antelación.

Sobre Minimal Marimba

Minimal Marimba es un proyecto artístico que explora la fusión entre la percusión clásica y la experimentación electrónica, creando un lenguaje sonoro único que respeta las raíces de la percusión mientras se adentra en territorios inexplorados.
